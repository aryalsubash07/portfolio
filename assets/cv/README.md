# CV Directory

Place your CV/Resume PDF file here.

## File Naming
- Name your CV file: `Subash_Aryal_CV.pdf`
- Or update the file path in `index.html` if you use a different name

## File Location
The CV download button in the portfolio will link to:
```
assets/cv/Subash_Aryal_CV.pdf
```

## Instructions
1. Export your CV as a PDF
2. Save it in this directory with the name `Subash_Aryal_CV.pdf`
3. The download button will automatically work

## Alternative
If you prefer a different filename, update the `href` attribute in `index.html`:
- About section: Look for `cv-download-btn` class
- Contact section: Look for `cv-link` class
